2.6 LISTS

2.6.1 What are HTML lists.

There are 3 type of lists in HTML

  • unordered lists,
  • ordered lists, and
  • definition lists.

2.6.2 Unordered Lists.

Unordered lists are created with the <ul> tag.

  • Item 1
  • Item 2
  • Item 3

2.6.3 Ordered Lists.

Ordered lists are create with the <ol> tag.

  1. First Item
  2. Second Item
  3. Third Item

2.6.4 Definition Lists.

Ordered lists are created with the <dl> tag.

Term 1
Definition 1
Term 2
Definition 2
Term 3
Definition 3

2.6.5 Nested Lists.

  • First Item.
  • Second item
    • Sub-Item 1
    • Sub-Item 2
  • Third Item.
  • First item.
  • Second item.
    1. Sub-item 1
    2. Sub-item 2
  • Third item.
