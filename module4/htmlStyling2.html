The id attribute

The id assigns a unique identifier to an element, which then can be used for several purposes, including:

  • unique identification lets you distinguish HTML elements from each other on the page.
  • styling allows to apply specific CSS styles to a single elment.
  • javascript manipulation, which lets you select and manipulate specific elements in the DOM.
