Font Properties

  • font-family defines the specific font name, such as Arial, Times New Roman or helvetica or a generic font family such as serif, sans-serif, or monospace
  • font-size it can be se using absolute value such as pixels (px) or points, or a relative value such as percentages(%) or em units(ems). The default value is normal
  • font-weight sets the weight
  • font-style defines the style of the font, such as normal (default value), italic, or oblique
